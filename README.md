> 📊 **A Python Data Analysis Project using Pandas, NumPy, and Matplotlib**

✨ 𝐀𝐛𝐨𝐮𝐭 𝐓𝐡𝐞 𝐏𝐫𝐨𝐣𝐞𝐜𝐭

This project performs **E-Commerce Sales Data Analysis** using Python libraries such as **Pandas**, **NumPy**, and **Matplotlib**.

The project cleans the dataset, handles missing values, removes duplicate records, calculates revenue, performs statistical analysis, identifies top-performing products, and visualizes sales trends through different charts.

---

✔ Import and analyze an E-Commerce sales dataset

✔ Handle missing values

✔ Remove duplicate records

✔ Calculate product revenue

✔ Perform statistical analysis

✔ Identify top-performing products

✔ Visualize data using charts


Dataset contains the following columns:

- Date
- Product
- Category
- Quantity
- Price
- Customer_Age
- Rating
- City

## ⚙️ 𝐅𝐞𝐚𝐭𝐮𝐫𝐞𝐬

### 🧹 Data Cleaning

- Fill missing values
- Replace missing product/category/city names
- Remove duplicate rows

### 💰 Revenue Calculation

Revenue is calculated using:

```
Revenue = Quantity × Price
```

### 📈 Statistical Analysis

The program calculates:

- Average Sales
- Median Revenue
- Standard Deviation
- Top Performing Products
